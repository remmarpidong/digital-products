# Etsy Shop Starter Pack

## Opening Your First Etsy Shop - Complete Guide

### Before You Start
- Have 10-20 product ideas ready
- Research pricing (search similar items)
- Prepare photos (natural light, white background)
- Set up PayPal or Etsy Payments

### Shop Setup Checklist
- [ ] Choose a memorable shop name
- [ ] Write a compelling about section
- [ ] Upload a professional banner
- [ ] Set shop policies (shipping, returns, processing time)
- [ ] Add 10 listings minimum before launching

## Product Photography Tips

### Camera Settings
- ISO: 100-400
- Aperture: f/5.6 - f/8
- White balance: Auto or daylight

### Lighting
- Natural light near window (north-facing best)
- Diffuse with white sheet or curtain
- Avoid harsh shadows

### Photo Order (for each product)
1. Main shot - straight on, full product
2. Detail shots - textures, features
3. In-use shot - showing context
4. Size reference - with ruler or coin

## SEO for Etsy

### Title Formula
[Keyword] + [Keyword] + [Style] + [Material] + [Use] + [Color]

Example: 
"Handmade Leather Journal, Vintage Notebook, Travel Diary, Gift for Her, Brown Leather Notepad"

### Tags Strategy
- Use all 13 tags
- Mix broad and specific tags
- Include synonyms
- Check search volume in Marmalead or eRank

### Description Template

```
[Brief intro - 2 sentences]

[MATERIALS - what's it made of?]

[SIZE - exact measurements]

[HOW TO ORDER - customization options]

[CARE INSTRUCTIONS]

[SHIPPING INFO]

[Thank you + shop info]
```

## Pricing Formula

```
Materials + Labor + Etsy Fees + Shipping + Profit = Price

Or use:
(Cost × 2) + Shipping = Wholesale
(Wholesale × 2) = Retail
```

## Launch Checklist
